## 🚀 Instalando o projeto

Para instalar, siga estas etapas:

Windows / Linux:
```
$ npm install 
```


## ☕ Rodando o projeto

Para rodar, siga estas etapas:

```
npm run serve
```

Para compilar, siga estas etapas:

```
npm run build
```
